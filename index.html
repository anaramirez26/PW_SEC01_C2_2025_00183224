Este libro es adecuado para cualquiera que tenga interés en aprender a desarrollar sus propias páginas web. No son necesarios conocimientos previos para aprender con este libro; lo único necesario es saber utilizar un ordenador y saber navegar por la Web.

Contenido del libro

En la primera parte del libro se trabajan conceptos generales necesarios para poder desarrollar páginas web: se explican conceptos de estructura física y organizativa de la Web; se muestra cómo se debe escribir la dirección de una página web (URL) y cómo se puede acceder a un sitio web.

En la segunda parte se trabaja HTML. Partiendo de la estructura mínima de una página web, se explican las etiquetas más comunes que se utilizan para definir el texto, los encabezados, los párrafos, las listas, los enlaces, las imágenes y las tablas.

En la tercera parte se trabaja CSS. Se explica cómo separar el contenido de la presentación, cómo aplicar estilos desde archivos externos y cómo controlar el diseño con propiedades de caja, tipografía, márgenes y disposición.
